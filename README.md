# Speech Recognition in Python using Google Speech API

This project demonstrates how to convert speech into text using Python and
Google Speech Recognition API.

## Features
- Converts WAV audio files into text
- Uses Google Speech Recognition API
- Simple and beginner-friendly AI project

## Installation
pip install SpeechRecognition pyaudio

## Usage
Place a clear .wav audio file in the project folder and run:
python speech_to_text.py

## Applications
- Voice assistants
- Speech-to-text systems
- Note-taking applications
- Accessibility tools

## Tech Stack
- Python
- Google Speech API
- SpeechRecognition Library
